Lazspel-2 version 2 by Raf20076, Poland, 2021

Lazspell-2 version 2 is an example of a speller which checks 
spelling in German, English, Spanish,
French, Italian, Portugues, 
Polish and Russian and underlines in red colour errors.

In this example TRichMemo component was used. 

Install TRichmemo from richmemo folder or TRichMemo from https://sourceforge.net/p/lazarus-ccr/svn/HEAD/tree/ 
Click Download Snapshot and uncompress lazarus-ccr-svn-r....zip
Go to folder lazarus-ccr-svn-r... -> components -> richmemo 
----------------------------------------------------------
To install richmemo component go to; 
Lazarus IDE menu->Package->Open Package File(.lpk)
Go to richmemo folder and open richmemopackage.lpk
In open window click Compile button. After Successful compilation go to
Lazarus IDE menu->Package->Open Package File(.lpk)
Go to richmemo folder and opend ide folder then open richmemo_design.lpk
In open window click Compile button. After Successful compilation click Use button and click Install option.
Lazarus ide will rebuild itself.
-----------------------------------------------------------
